other possible names: nownow? storiograph?

# getting things running

First install [node](http://nodejs.org).

Next, in the project root, do:

```
npm install
```

Then for development, do:

```
npm run watch
```

Or to build for production, do:

```
npm run build
```

To start the server, do:

```
npm start
```
